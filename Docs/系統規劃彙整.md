# 矩形轉扇形圖系統 — 規劃彙整
*截至 2026-04-20*

---

## 一、系統定位

- 協助使用者將矩形設計圖轉換為紙杯扇形展開圖（印刷用）
- 交付方式：**網頁版**
- 部署環境：虛擬機或雲端（NAS 為備選）
- Prompt 完全隱藏於後端，使用者不可見
- UI 風格：輕鬆、休閒、活潑，貼近使用者

---

## 二、核心流程

```
使用者選擇杯型
      ↓
系統自動帶入：比例、R、r、θ、Canva 建議尺寸
      ↓
Stage 1：選擇輸入方式
├── 1A：AI 生成矩形圖
│   使用者填入：主題、風格、主要文字
│   後端組裝：固定規則 Prompt + 使用者輸入 → AI API
│   輸出：矩形圖 PNG
│
└── 1B：Canva 製作後上傳
    系統顯示：建議 Canva 尺寸（如 2294×1029 px）
    使用者上傳後 → 後端驗證尺寸與比例
    通過 → 矩形圖 PNG
      ↓
使用者確認矩形圖
      ↓
Stage 2：後端 Python Polar Warp（不需要 AI）
自動帶入 R、r、θ → 數學極座標轉換 → 扇形圖 PNG
      ↓
提供分別下載：矩形圖 PNG + 扇形圖 PNG（兩個獨立檔案）
```

### AI 使用範圍

| 階段 | 需要 AI | 說明 |
|------|:-------:|------|
| Stage 1A | ✅ | 生成矩形圖視覺 |
| Stage 1B 上傳驗證 | ❌ | 後端數學驗證 |
| Stage 2 轉換 | ❌ | Python Polar Warp，零 AI 費用 |

## 四、Prompt 管理

### 結構
Stage 1A Prompt 為模板設計：

```
固定段落（管理員可編修）
├── 背景規則（純白）
├── Logo 位置規則（右上）
├── 文字排版規則（水平居中）
└── 其他設計規範

動態帶入（系統自動）
├── 畫布比例 → 由杯型資料庫自動計算
├── 主題     → 使用者輸入
├── 風格     → 使用者輸入
└── 主要文字 → 使用者輸入
```

> Stage 2 不是 Prompt，是後端 Python 程式，不進入 Prompt 管理系統。

### 版本控制
```
prompt_versions 表
├── id
├── segment_name     （段落名稱）
├── segment_order    （排列順序）
├── content          （Prompt 內容）
├── version          （版本號）
├── is_active        （是否生效）
├── edited_by        （修改者）
└── created_at
```
- 每次修改新增一筆，不覆蓋
- 系統管理員與數據管理員可回溯版本

## 五、杯型數據管理

### 資料來源
**紙杯扇環幾何計算表.pdf**（數據非最終值，持續優化中）

### 命名規則
| 系列 | 格式 | 範例 |
|------|------|------|
| 單層 | XXoz_冷熱杯 | 8oz_冷熱杯 |
| 雙層（OW） | XXoz_雙層杯 | 8oz_雙層杯 |

### 現有杯型（共 10 種）

| 杯型 | R (mm) | r (mm) | θ (°) | W:H 比例 | Canva 尺寸 (px) |
|------|-------:|-------:|------:|:--------:|:--------------:|
| 8oz_冷熱杯 | 326.18 | 238.56 | 41.17 | 2.230 | 2294×1029 |
| 12oz_冷熱杯 | 369.95 | 263.51 | 40.96 | 2.103 | 2588×1231 |
| 16oz_冷熱杯 | 426.61 | 296.43 | 35.55 | 1.805 | 2605×1443 |
| 20oz_冷熱杯 | 502.33 | 343.30 | 30.12 | 1.528 | 2610×1708 |
| 22oz_冷熱杯 | 600.36 | 436.32 | 25.11 | 1.496 | 2610×1745 |
| 24oz_冷熱杯 | 512.35 | 346.54 | 31.12 | 1.540 | 2749×1785 |
| 8oz_雙層杯 | 357.69 | 280.67 | 38.26 | 2.534 | 2344×925 |
| 12oz_雙層杯 | 397.51 | 300.63 | 38.99 | 2.325 | 2653×1141 |
| 16oz_雙層杯 | 463.75 | 342.79 | 33.57 | 1.976 | 2678×1356 |
| 20oz_雙層杯 | 551.31 | 402.41 | 28.04 | 1.660 | 2671×1609 |

### 資料庫設計
**只儲存 R、r、θ，其餘後端即時計算：**

```
cup_sizes 表
├── id
├── cup_name         （如 "8oz_冷熱杯"）
├── cup_series       （cold_hot / double_wall）
├── R_mm             ← 輸入
├── r_mm             ← 輸入
├── theta_deg        ← 輸入
├── is_active
├── note             （備註）
├── edited_by
└── created_at

cup_size_versions 表（變動歷史）
├── id
├── cup_size_id
├── snapshot_json    （完整數據快照）
├── edited_by
└── created_at
```

**後端即時計算項目：**
- 厚度 t = R − r
- 外弧長 = R × θ(rad)
- 內弧長 = r × θ(rad)
- 杯口直徑 = 外弧長 / π
- 杯底直徑 = 內弧長 / π
- 外接框 W = 2R × sin(θ/2)
- 外接框 H = R − r × cos(θ/2)
- 比例 W:H = W / H
- Canva 建議尺寸 = W(mm) × 10 : H(mm) × 10

- Gemini API


### 共用 Key 保護
- 每日使用次數上限
- 每用戶使用次數上限
- 可限制匿名用戶不得使用共用 Key
- Key 加密儲存，顯示時遮罩（sk-****1234）

- 主要來源選擇
- 各來源 API Key / 本地 URL 設定
- Stage 1A 指定 Model
- 連線測試按鈕

## 七、檔案管理

### 命名格式
```
{杯型名稱}_{圖型}_{日期}_{時分秒}.png

範例：
8oz_冷熱杯_矩形圖_20260420_143052.png
8oz_冷熱杯_扇形圖_20260420_143052.png
```
- 時間：伺服器時間
- 空格：統一改為底線
- 同一次操作兩個檔案時間戳記相同

### 保留規則
| 項目 | 規則 |
|------|------|
| 保留筆數 | 每位用戶僅保留最後一次產生的檔案 |
| 計算基準 | 以產生時間計算，到期即清除 |
| 保留天數 | 系統管理員可設定 |
| 匿名用戶 | 不保留，需立即下載 |
| 重新下載 | 未清除前可重新下載最後一筆 |

### 下載記錄
```
download_records 表
├── id
├── user_id
├── cup_size_version_id  （綁定當時數據版本）
├── prompt_version_id    （綁定當時 Prompt 版本）
├── rectangle_file       （矩形圖路徑）
├── fan_file             （扇形圖路徑）
└── created_at           （清除依據）
```
